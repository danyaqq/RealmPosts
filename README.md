## Инструкция по установке
1. Скачать приложение https://github.com/danyaqq/RealmPosts.git
2. Открыть терминал
3. Перейти в папку приложения (cd /../AnotherRealm)
4. Вбить в терминале pod install
5. Запустить файл AnotherRealm.xcworkspace
